github
-on windows: using git bash
-clone repository
-placed in folder
-file moved into repository (only using master branch)
-use "git add ." (or could use "git add file_name")
-"git status" checks the changes to be committed
-git commit
  -press "i" to enter inline insert mode
  -press "esc" to exit insert mode
  -type ":x!" (cursor is at the bottom)
  -press "enter" to save
-"git push" pushes the changes committed
